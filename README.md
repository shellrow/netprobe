# netprobe
Cross-Platform Network Probe Library. Written in Rust.

## Features
- traceroute
    - [x] IPv4 UDP
    - [x] IPv6 UDP
- ping
    - [x] IPv4 ICMPv4
    - [x] IPv6 ICMPv6
    - [x] IPv4 UDP
    - [x] IPv6 UDP
    - [x] IPv4 TCP
    - [x] IPv6 TCP

## Usage
Add `netprobe` to your dependencies
```
[dependencies]
netprobe = "0.1"
```
